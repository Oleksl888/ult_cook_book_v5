API

Access

It is free to access certain endpoints of the API. Without access key you can request recipes, ingridients and feedback endpoints. If you decide to use full access, you have to request an access key. Using this key, you can add feedback to the website and individual recipes. To access user information and make changes to the website you must have a token with admin priveleges set.

Base endpoint and example

You can use the following endpoints:
  • https://ultcookbook.herokuapp.com//api/recipe(/id:optional)
  • https://ultcookbook.herokuapp.com//api/ingridients(/id:optional)
  • https://ultcookbook.herokuapp.com//api/register
  • https://ultcookbook.herokuapp.com//api/login
  • https://ultcookbook.herokuapp.com//api/users(/id:optional)
  • https://ultcookbook.herokuapp.com//api/feedback/
  • https://ultcookbook.herokuapp.com//api/recipe/feedback(/id:optional)
Form-data and json requests are accepted. Access token required for most POST requests.
